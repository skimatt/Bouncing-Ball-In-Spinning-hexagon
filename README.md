# Bouncing Ball in a Spinning Hexagon

Proyek ini adalah simulasi Python yang menampilkan bola memantul di dalam heksagon yang berputar. Bola dipengaruhi oleh gravitasi dan gesekan, serta memantul secara realistis dari dinding heksagon yang berputar.

## Persyaratan

- Python 3.x
- Pygame

## Instalasi

1. **Clone repositori ini**:
   ```bash
   git clone https://github.com/username/repository-name.git
   cd repository-name
   ```

2. **Instal Pygame**:
   Jika Anda belum memiliki Pygame, Anda dapat menginstalnya menggunakan pip:
   ```bash
   pip install pygame
   ```

## Menjalankan Program

Setelah menginstal Pygame, Anda dapat menjalankan program dengan perintah berikut:
```bash
python bouncing_ball.py
```

## Deskripsi Program

- **Heksagon Berputar**: Heksagon berputar secara terus-menerus di tengah layar.
- **Bola Memantul**: Bola bergerak di dalam heksagon dan memantul dari dindingnya. Bola dipengaruhi oleh gravitasi dan gesekan.
- **Kontrol**: Program akan berjalan secara otomatis. Untuk keluar, tutup jendela program atau tekan tombol `ESC`.

## Struktur Kode

- **`bouncing_ball.py`**: File utama yang berisi logika simulasi dan rendering menggunakan Pygame.
- **`README.md`**: File ini, berisi informasi tentang proyek.
